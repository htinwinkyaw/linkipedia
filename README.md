## linkipedia
**linkipedia** is a place to collect the users' social links and share with others with ease.

---

### Techstacks
- NextJS
- ReactJS
- TypeScript
- TailwindCSS
- DaisyUI

---

### Installation
`
  git clone https://github.com/htinwinkyaw/linkipedia.git<br>
  cd linkipedia<br>
  npm install
`
